🔴 Critical Priority

Do First

Immediate Security Checks

  • ✓
    Check iPhone Configuration Profiles
    Settings → General → VPN & Device Management. Remove ANY profiles you did not install.
  • ✓
    Review Google Third-Party Access
    Remove all applications you do not recognise or no longer use.
    myaccount.google.com/permissions
  • ✓
    Check Gmail Forwarding Rules
    Gmail Settings → Forwarding and POP/IMAP. Disable any forwarding to addresses you don't control.
  • ✓
    Review Gmail Filters
    Gmail Settings → Filters. Delete any filters you did not create, especially those that auto-delete or forward.
  • ✓
    Check Apple Account Trusted Devices
    Settings → [Your Name] → scroll down to see devices. Remove any you don't recognise.
  • ✓
    Review Location Sharing
    Settings → Privacy & Security → Location Services → Share My Location. Remove anyone suspicious.

📱 iPhone Security

High Priority

Device Hardening

  • ✓
    Enable Stolen Device Protection
    Settings → Face ID & Passcode → Stolen Device Protection → Always
  • ✓
    Enable Advanced Data Protection
    Settings → [Your Name] → iCloud → Advanced Data Protection. Enables end-to-end encryption.
  • ✓
    Review iMessage Text Forwarding
    Settings → Messages → Text Message Forwarding. Remove any devices you don't own.
  • ✓
    Audit Third-Party Keyboards
    Settings → General → Keyboard → Keyboards. Remove any keyboards you did not install.
  • ✓
    Review App Location Permissions
    Settings → Privacy & Security → Location Services. Set most apps to "While Using" or "Never".
  • ✓
    Check for Unknown AirTags
    Open Find My app → Items tab. Look for any items you do not own.
  • ✓
    Enable Mail Privacy Protection
    Settings → Mail → Privacy Protection → Protect Mail Activity ON

🔐 Google Workspace Security

High Priority

Account Protection

  • ✓
    Complete Google Security Checkup
    Follow all recommendations in the security checkup wizard.
    myaccount.google.com/security-checkup
  • ✓
    Review Connected Devices
    Sign out of any devices you don't recognise.
    myaccount.google.com/device-activity
  • ✓
    Configure Hardware Security Key
    Add a YubiKey or Titan key for strongest 2FA protection.
    myaccount.google.com/signinoptions/two-step-verification
  • ✓
    Generate Backup Codes
    Print and store securely offline. Do not store digitally.
  • ✓
    Enable Enhanced Safe Browsing
    Provides real-time protection against malicious sites.
    myaccount.google.com/security
  • ✓
    Review Recent Security Activity
    Check for any suspicious login attempts or changes.
    myaccount.google.com/notifications
  • ✓
    Update Password (20+ Characters)
    Use a password manager to generate and store a strong unique password.

🌐 Network Security

Medium Priority

Network Hardening

  • ✓
    Configure Secure DNS
    Use Cloudflare (1.1.1.1) or Quad9 (9.9.9.9) instead of ISP DNS.
  • ✓
    Enable Private Wi-Fi Address
    Settings → Wi-Fi → [Network] → Private Wi-Fi Address ON
  • ✓
    Review Router Admin Password
    Change from default. Use strong unique password.
  • ✓
    Update Router Firmware
    Check manufacturer website for latest firmware.
  • ✓
    Enable macOS Firewall
    System Preferences → Security & Privacy → Firewall → Turn On

👤 Other Accounts

Medium Priority

Connected Services

  • ✓
    Secure GitHub Account
    Enable 2FA, review sessions and authorized apps.
    github.com/settings/security
  • ✓
    Secure Dropbox Account
    Enable 2FA, review linked devices and apps.
    dropbox.com/account/security
  • ✓
    Secure PayPal Account
    Enable 2FA, review login activity.
    paypal.com/myaccount/settings/
  • ✓
    Review Opera Account Sessions
    Terminate unknown sessions.
    auth.opera.com/account/edit-profile#session-section
  • ✓
    Install Password Manager
    Use 1Password or Bitwarden. Generate unique passwords for every account.
