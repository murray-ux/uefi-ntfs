## About

Auth Portal implements authentication patterns from MediaWiki/Wikimedia Commons:

- **Enhanced Password Reset (EPR)** - Require both username AND email
- **Privacy-First Design** - Never reveal account existence
- **Multi-Language Support** - 14+ languages
- **Accessibility** - Theme toggle, text sizing, collapsible sections
- **MediaWiki-Compatible** - Salted password hashing, maintenance scripts
